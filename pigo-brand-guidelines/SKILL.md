# Pigo Inc. Brand Guidelines

## Description
用於確保所有產出符合 Pigo Inc. 的品牌身分。當用戶要求撰寫文案、設計介面或討論品牌策略時，必須參考此規範。

## Instructions

### 1. 品牌核心 (Core Identity)
- **品牌使命：** 透過極簡科技簡化複雜生活。
- **品牌個性：** 專業但幽默 (Witty Professional)、誠實、充滿活力。

### 2. 視覺規範 (Visual Standards)
- **核心色彩：** - Pigo Orange (#FF6B00) - 主色，用於行動呼籲。
  - Deep Slate (#2F3E46) - 文字與背景色。
- **字體建議：** 標題使用 Inter Bold，內文使用 Helvetica Neue。
- **標誌使用：** 標誌周圍必須保持至少 20px 的留白。

### 3. 語調與風格 (Tone of Voice)
- **Do:** 使用主動語態、在結尾適度加入幽默感、稱呼用戶為「夥伴」。
- **Don't:** 嚴禁使用過度艱澀的術語、禁止使用過於嚴肅的官腔。
- **格式規範：** 列表優先，避免長篇大論。

### 4. 特定術語 (Glossary)
- 使用 "Pigo Smart Link" 而非 "The connection feature"。
- 產品名稱首字母必須大寫。
